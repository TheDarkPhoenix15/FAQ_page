FAQs

  • What is Frontend Mentor, and how will it help me?

    • Frontend Mentor offers realistic coding challenges to help
    developers improve ther frontend coding skills with projects in
    HTML, CSS and JavaScript. It's suitable for all levels and ideal for porfolio building.

  • Is Frontend Mentor free?

    • Frontend Mentor offers a free package for beginners and
    a very excellent deal on the premium offer which gives people
    a chance on polishing their skills and making an excellent portfolio.

  • Can I use Frontend Mentor projects in my portfolio?

    • You can decide to utilize the projects you complete in a
    portfolio that is provided by frontend mentor. You can also
    decide to include it in any other portfolio platform of your choice.

  • How can I get help if I'm stuck on a challenge?

    • Frontend mentor offers a solution in the form of a discord
    server in the case that you get stuck. Here, you can chat with
    seasoned experts and get feedback on any project you are undertaking.
